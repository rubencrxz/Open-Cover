# 📦 Open-Cover
OpenCover Insured Vaults is a covered metavault: an upgradeable ERC-4626 vault that wraps a single underlying ERC-4626 vault. Deposits and redemptions are asynchronous (ERC-7540): users submit requests, a trusted keeper settles them in discrete epochs, and users then claim shares or assets. The vault uses internal asset accounting instead of balance checks to avoid donation and inflation risks. A configurable annual premium is streamed from settled assets to a collector, with a per-vault cap and a global 25% maximum. It follows ERC-7575 single-asset semantics (share token is the vault itself) and supports synchronous cancellation of pending deposits and unsettled redemptions (ERC-7887 subset). Operator delegation is disabled; all operations require controller, owner, and receiver to be the same address. If the keeper stalls, users can permissionlessly settle matured redemptions after 24 hours. The implementation is UUPS-upgradeable, uses ERC-7201 namespaced storage, and targets Ethereum, Base, Arbitrum, and Optimism.
